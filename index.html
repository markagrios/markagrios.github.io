Current Research —

I'm very lucky to be doing research as an undergraduate. I'm working with two professors to study biological neural networks using simplicial homology. I'm using Brian simulator (yes Brian not Brain), a neuron spiking simulator package for Python, to build a Hindmarsh-Rose model for a small neural network that we can then look at using techniques from algebraic topolgy.

PushBuddy —

My freshman year of college, my friends and I competed in VT Hacks III and won second place for our hack. We wanted to tackle the confusion and frustration that comes with all the different cloud storage services out there and PushBuddy was born. I designed the website :D

I'm using mean curvature flow over smooth topologies as an optimizing mechanic and continuous threshold biased neurons. I'm trying to model LeToLi's learning after a single organisms brain solving local problems not genetic algorithms that model the adaptations of a species like other machine learning programs.
